# Data policy

This directory is reserved for small, intentional input data or metadata needed
to reproduce the experiment. The current Stages 1-11 pipeline generates its
graphs in memory and requires no input dataset.

Do not commit generated experiment results, large datasets, private data,
credentials, or the local methodology PDF here. Generated outputs belong in the
ignored `results/` or `outputs/` directories and should be reviewed before any
separate publication decision.
